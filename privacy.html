Политика конфиденциальности

Дата последнего обновления: 20 мая 2026 г.

Настоящая Политика конфиденциальности объясняет, как сервис AI Photo («Сервис», «мы», «наш») обрабатывает информацию при использовании вами нашего сайта и Telegram-бота.

1. Какие данные мы обрабатываем

Сервис может обрабатывать следующие данные:

- Изображения, загруженные пользователем для генерации AI-фото.
- Идентификатор пользователя в Telegram (Telegram User ID).
- Технические данные: IP-адрес, user-agent, временные метки запросов.
- Данные аналитики: информация о посещении страниц сайта (через Google Analytics / Яндекс Метрику).

2. Как мы используем данные

- Загруженные изображения используются исключительно для выполнения запроса пользователя (генерация AI-фото) и временно хранятся в течение технически необходимого периода.
- Telegram User ID используется для идентификации пользователя в боте и предоставления доступа к функциям сервиса.
- Данные аналитики используются для улучшения работы сервиса и анализа трафика.

3. Telegram-бот

Основная функциональность сервиса реализована через Telegram-бота. При использовании бота применяются политика конфиденциальности Telegram и настоящая Политика. Данные обрабатываются в соответствии с условиями Telegram Platform.

4. Публикация результатов

Результаты генерации НЕ публикуются автоматически. Пользователь самостоятельно принимает решение о публикации изображения в публичных разделах форума. До момента публикации изображение доступно только пользователю.

5. Передача данных третьим лицам

Мы не продаём и не передаём пользовательские данные третьим лицам, за исключением случаев, предусмотренных законодательством. Сервис может использовать сторонние инструменты аналитики (Google Analytics, Яндекс Метрика), которые обрабатывают обезличенные данные о посещениях.

6. Хранение данных

Данные хранятся в течение срока, необходимого для предоставления функциональности сервиса. Пользователь может запросить удаление своих данных, обратившись через Telegram-бота или контактную информацию ниже.

7. Файлы cookie

Сайт может использовать технические cookie для обеспечения корректной работы. Cookie аналитики используются для сбора обезличенной статистики посещений.

8. Контакты

По всем вопросам, связанным с обработкой данных, вы можете обратиться через Telegram-бота или форум сервиса.
